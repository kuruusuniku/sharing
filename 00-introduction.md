# 0. はじめに

## AIの進化が閾値を超えた

AI開発はいま急速に進んでおり、2大巨頭は **Anthropic** と **OpenAI** です。

それぞれが開発者向けのAIコーディングツールをリリースしています：

| 企業 | ツール | 最新モデル |
|------|--------|-----------|
| **Anthropic** | Claude Code | Opus 4.7 |
| **OpenAI** | Codex | GPT-5.5 |

Google（Gemini）のモデルはまだ実務で使える閾値を超えておらず、仕事で本格活用するなら基本的にこの2社のツールが選択肢になります。

## 日常利用と仕事利用の違い

日常的な相談ごと（調べもの、文章作成、アイデア出し）であれば ChatGPT や Gemini でも十分です。

ただし、**仕事として成果物を出す**となると話が変わります。コードを書く、資料を作る、タスクを管理する——こうした業務にはAnthropicかOpenAIのツールが必要です。

## なぜ Claude Code なのか

Claude Code と Codex はどちらも優れていますが、**現時点でのメインストリームは Claude** です。

まずは **Claude Desktop（デスクトップアプリ）** から始めるのもアリです。チャット形式で Claude と対話でき、日常業務にすぐ使えます。

ただし、AIを高度に活用する——ファイル操作、コマンド実行、複数タスクの並列処理——には **Claude Code（CLI）** が最適です。この資料では Claude Code を中心に解説します。

## プランについて

| プラン | 月額 | 所感 |
|--------|------|------|
| Pro | $20 | 入門には十分。ただし利用制限にすぐ当たる |
| **Max** | **$100** | 満足に使うにはこのプランが必要 |
| Max 20x | $200 | ヘビーユーザー向け。制限がさらに緩和 |
| Team | $30/人 | 個人の Max プランの方がコスパが良い |

正直なところ、Max $100 プランに入らないと仕事で満足に使えない面はあります。Team プランは管理機能が加わる分、個人の利用量としては Max の方がお得です。

## 賢いモデルだけでは足りない

AIモデルがどれだけ賢くても、それ単体では力を発揮しきれません。

**モデルの能力を引き出すには、それを組み込む「仕組み」が必要です。**

- どういう順番でタスクを処理するか
- 複数のAIをどう協調させるか
- 成果物の品質をどうチェックするか

これが、後の章で紹介する **ワークフロー**（[2章](02-agent-patterns)）や **オーケストレーションツール**（[4章](04-custom-orchestration)）と呼ばれるものです。

この資料を通じて、AIを「使う」だけでなく「仕組みとして活用する」方法を共有できればと思います。
