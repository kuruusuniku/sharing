# 11. OpenClaw — 個人用AIアシスタント

ここまでの章では、コーディングや業務タスクにフォーカスしてきました。この章では視点を変えて、**日常生活全体をAIでアシストする**ツールを紹介します。

## OpenClaw とは

[OpenClaw](https://openclaw.ai/) は、オープンソースの個人用AIアシスタントです。自分のPC上で動作し、複数のチャットアプリ・サービスと連携して、日常のあらゆるタスクを自動化できます。

**一言でいうと：** 「自分専用の Siri / Google Assistant を、好きなLLMで、好きなサービスと繋げて作れる」ツールです。

## 何ができるのか

### チャットアプリから操作

普段使っているチャットアプリから直接AIに指示を出せます。

- WhatsApp
- Telegram
- Discord
- Slack
- Signal
- iMessage

専用アプリを開く必要がなく、**いつものチャットで「あれやっといて」と頼むだけ**で動きます。

### サービス連携（50以上）

| カテゴリ | 連携サービス例 |
|---------|--------------|
| **メール** | Gmail |
| **コード** | GitHub |
| **ドキュメント** | Obsidian |
| **スマートホーム** | Hue |
| **音楽** | Spotify |
| **カレンダー** | Google Calendar |

### システム操作

- ファイルの読み書き
- シェルコマンドの実行
- ブラウザ操作（Webサイト閲覧、フォーム入力、データ抽出）

### 永続メモリ

ユーザーの好み・文脈を記憶し、使うほど個別化されたアシスタントに成長します。

## 他のツールとの違い

| | ChatGPT / Claude Desktop | OpenClaw |
|--|--------------------------|----------|
| **LLM** | 固定（各社のモデル） | 自由に選べる（Claude, GPT, ローカルLLM等） |
| **データ** | クラウドに送信 | ローカルに保存 |
| **連携** | プラグイン/MCP経由 | 50以上のサービスに直接連携 |
| **操作方法** | 専用アプリを開く | 普段のチャットアプリから |
| **カスタマイズ** | 限定的 | 完全に自由（オープンソース） |
| **コスト** | 月額サブスク | 無料（LLMのAPIコストのみ） |

## セットアップ

macOS / Linux / Windows に対応。ワンライナーでインストールできます。

```bash
# インストール（公式サイトのコマンドに従う）
curl -fsSL https://openclaw.ai/install.sh | sh
```

LLMのバックエンドは自分で選択：
- **Claude API** — 高品質な応答
- **OpenAI API** — GPT系モデル
- **ローカルLLM（LM Studio等）** — コストゼロ・プライバシー完全保護

## 活用例

```
# Telegramから
「明日の会議の議題をGmailから拾ってまとめて」

# Slackから
「GitHubのPR #42 のレビューコメントを要約して」

# iMessageから
「今日の予定を確認して、夕方空いてたら歯医者を予約して」

# Discordから
「Obsidianの先週のメモを検索して、プロジェクトAの進捗をまとめて」
```

## なぜ紹介するのか

Claude Code や MCP は**仕事のツール**です。一方 OpenClaw は**生活全体のツール**です。

AIの活用は仕事だけに閉じる必要はありません。日常のメール処理、スケジュール管理、情報整理——こうした雑務をAIに任せることで、本来集中すべき仕事にもっと時間を使えるようになります。
