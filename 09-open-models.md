# 9. オープンLLMとローカル実行

重みが公開されており、自分のマシンで動かせるLLMの世界です。クラウドに依存しない、プライバシー重視、コスト無制限という利点があります。

## オープンLLMの勢力図

### 中華勢 — 急速に台頭

| モデル | 開発元 | 特徴 |
|--------|--------|------|
| **Qwen 3** | Alibaba | コーディング・推論に強い。日本語も良好。軽量〜大型まで幅広いサイズ展開 |
| **DeepSeek V3 / R1** | DeepSeek | 推論特化モデル（R1）が話題に。コスト効率が非常に高い |

中華勢の強みは**コスト効率とサイズ展開の幅広さ**。特に Qwen は 0.6B〜235B まで揃っており、用途に合わせて選べます。

### 西側諸国

| モデル | 開発元 | 特徴 |
|--------|--------|------|
| **Llama 4** | Meta | オープンLLMの代表格。研究・商用ともに利用可能 |
| **Mistral Large** | Mistral AI | フランス発。欧州言語に強い。軽量で高性能 |
| **Gemma 2** | Google | 軽量で研究用途に最適。Google の技術を反映 |
| **Phi-4** | Microsoft | 小型ながら高性能。エッジデバイス向け |

### 日本語特化

| モデル | 開発元 | 特徴 |
|--------|--------|------|
| **ELYZA** | ELYZA（東大発） | 日本語に特化したLlama ベースモデル |
| **PLaMo** | Preferred Networks | 日本語性能に注力。国産LLM |
| **Japanese StableLM** | Stability AI Japan | 日本語対応の安定した汎用モデル |
| **Swallow** | 東工大 | 学術研究ベースの日本語LLM |

日本語特化モデルは、日本語での自然さ・正確さでは汎用モデルを上回ることがあります。ただし、コーディング能力はQwen や Llama の方が上です。

## ローカル実行ツール

### LM Studio

**おすすめ度：最も手軽**

GUIでモデルをダウンロード・実行できるデスクトップアプリ。技術知識がなくても使えます。

- モデルの検索・ワンクリックダウンロード
- チャットUIで即座に試せる
- OpenAI互換のローカルAPIサーバーを立てられる（他ツールとの連携に便利）
- macOS / Windows / Linux 対応

```
LM Studio を起動
 → モデルを検索（例: "qwen3"）
 → ダウンロード
 → チャット or ローカルAPIサーバーを起動
```

### OpenCode

CLIベースのAIコーディングツール。ローカルLLMをバックエンドに使えます。

- Claude Code のようなCLI体験をローカルLLMで実現
- LM Studio のローカルAPIと組み合わせて使う
- オフラインでのコーディング支援が可能

## 必要なマシンスペック

ローカルLLMを快適に動かすには、**メモリ（RAM/VRAM）が最重要**です。

| モデルサイズ | 必要メモリ目安 | 体感 |
|-------------|--------------|------|
| 1B〜3B | 4GB〜 | 軽量。スマホでも動く。品質は限定的 |
| 7B〜8B | 8GB〜 | 実用最低ライン。簡単なタスクなら十分 |
| 14B〜27B | 16GB〜32GB | 日常利用に十分な品質 |
| 70B〜 | 48GB〜 | クラウドモデルに迫る性能。要ハイスペック |

### Mac mini がおすすめ

ローカルLLM用途で**Apple Silicon の Mac mini は最もコスパが良い選択肢**です。

**理由：**
- **ユニファイドメモリ** — CPUとGPUでメモリを共有。VRAM不足に悩まされない
- **Metal対応** — Apple GPU でLLM推論が高速
- **省電力** — 24時間つけっぱなしでも電気代が低い
- **コンパクト** — サーバーとして置いても場所を取らない

| 構成 | 価格目安 | 動かせるモデル |
|------|---------|--------------|
| Mac mini M4 16GB | 約10万円 | 7B〜14B が快適 |
| Mac mini M4 Pro 24GB | 約25万円 | 14B〜27B が快適 |
| Mac mini M4 Pro 48GB | 約35万円 | 27B〜70B も実用的 |

**おすすめ構成：** M4 Pro 24GB以上。Qwen3 27B が快適に動き、コーディング支援や設計レビューに十分使えます。

## ローカルLLMの使いどころ

| ユースケース | おすすめ |
|-------------|---------|
| 機密性の高いコードの分析 | ローカルLLM（データが外に出ない） |
| オフライン環境での作業 | ローカルLLM |
| 大量の繰り返し処理 | ローカルLLM（APIコストゼロ） |
| 最高品質の出力が必要 | クラウドモデル（Claude Opus / GPT） |
| 設計・レビューの壁打ち | ローカルLLM（気軽に何度でも） |
